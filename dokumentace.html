Úvod

Téma semestrální práce z předmětu Značkovací jazyky je Konfigurace neboli nastavení aplikace, tedy konkrétně smyšlené počítačové hry. Konfigurační soubor obsahuje jen základní nastavení hry - obecné nastavení, nastavení grafiky, zvuku a ovládání.

Obsahem práce je:

  • XML soubor
  • XML schéma
  • XSLT soubor
  • Transformace do JSON
  • Tato dokumentace v HTML

Metodika

Celá semestrální práce byla tvořena ručně bez konvertorů v prostředí NetBeans. První jsem vytvořila soubor XML, který jsem následně přepsala do XSD a validitu jsem zkontrolovala pomocí NetBeans. Poté jsem vytvořila soubor XSLT a spolu s XML jsem ho transformovala do JSON. Jako poslední jsem vytvořila dokumentaci v HTML, jejíž validitu jsem zkontrolovala validátorem W3C.

Použité aplikace

Pro tvorbu semestrální práce jsem využila vývojového prostředí NetBeans IDE 8.2. a prohlížeče Google Chrome a Microsoft Edge.

Použité webové stránky

Jako podporu jsem použila prezentace na Moodlu z předmětu Značkovací jazyky a webovou stránku w3schools. Pro validaci HTML souboru jsem využila stránky Validator W3C.

Popis transformace

Transformace do JSON (JavaScript Object Notation) je provedena pomocí XML a XSLT souboru. Je zde použitý cyklus for-each, který vybere a odešle informace o hráči a nastavení hry.
